# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Data Science Internship**.

The objective of this project is to build a Machine Learning model that can accurately classify Iris flowers into one of three species based on their flower measurements.

The dataset used is the famous Iris Dataset available in the Scikit-learn library.

---

## 🎯 Objective

To classify Iris flowers into:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

using Machine Learning algorithms.

---

## 📂 Dataset

- Source: Scikit-learn Iris Dataset
- Total Samples: 150
- Features: 4
- Classes: 3

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 📈 Model Evaluation

The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report

Random Forest achieved the highest accuracy and was selected as the best-performing model.
